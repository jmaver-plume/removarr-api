# Removarr (Rust)

This project is removarr rewritten from Node.js to Rust as a learning exercise.


## APIs needed

```
# Webhooks
POST /removarr/webhook/sonarr
POST /removarr/webhook/radarr


# Login
GET /removarr/login
POST /removarr/login
POST /removarr/logout


# Settings
GET /removarr/settings
PUT /removarr/settings
PATCH /removarr/settings


# API
POST /removarr/resync
```
